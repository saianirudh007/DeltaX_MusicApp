# DeltaX Music Application(Minigic)
      
I have Developed a Music Web Application using .NET Framework. Below are the details and Screenshots of the Application. The technologies that I used

- Frontend
  - HTML
  - CSS
  - BootStrap
  - JS
  - Ajax

- Backend
  - ASP.NET
  - Entity Framework
  - SQL Server Database
